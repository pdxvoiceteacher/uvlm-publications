# Triadic Observation Contract

TRIADIC-OBSERVATION-CONTRACT-DESIGN-00 is an active-design-only, consent-bounded governed-attention contract. Governed Attention Precedes Governed Intelligence: the system must govern how the receipt is born before any publication surface may describe stronger runtime behavior.

The contract declares observation scope, resolution, purpose, consent, retention, replay, disclosure, federation, recovery rights, reciprocity budget, and non-authority boundaries. It does not change runtime behavior, emit mode-shift receipts, perform user recovery actions, authorize surveillance, export traces, federate PMR, write memory, admit Atlas memory, call providers or networks, deploy, release product, train models, skip review, certify truth, or grant final-answer or accepted-evidence authority.
