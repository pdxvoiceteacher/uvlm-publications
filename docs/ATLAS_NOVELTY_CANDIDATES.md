# Atlas Novelty Candidates

This document describes the intermediate artifact used when Sophia routes a novel track for future Atlas preservation.

## Artifact
`atlas_novelty_candidate.json`

## Purpose
Preserve:
- track id
- model/node/origin
- novel tokens
- source-evidence readiness
- coherence metrics
- deltas
- candidate output

## Design stance
This artifact is:
- not yet a permanent Atlas prior
- not yet canonized
- a candidate for future phaselock validation and persistence
