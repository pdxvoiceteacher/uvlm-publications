governed artifacts after Sophia and performs no model or network activity.

digest to manifest file bytes. Atlas now distinguishes compact canonical JSON
digests from file-byte digests: hashes establish identity, not truth.

candidate identities plus Sophia 1.1 producer, route, parent-list, digest,
nonauthority, and side-effect ceilings. PASS maps to
`retain_for_human_review` / `publication_blocked_pending_human_review`;
HOLD maps to `quarantine` / `do_not_publish`; REJECT maps to
`rejected` / `do_not_publish`.

The static review presents current request, source, candidate claims and
citations, Sophia findings, and Atlas posture. Atlas posture is not memory,
PMR, canonization, publication, DOI, Crossref, deployment, or release
authority; human final authority remains binding.

The future Atlas human-dialogue node is deferred, not cancelled, and must
remain subordinate to the deterministic packet. One real three-repository
acceptance run remains the next product gate.
