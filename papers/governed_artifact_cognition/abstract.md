# Abstract

This paper introduces governed artifact cognition as a reproducible audit-lab pattern for AI-assisted candidate review. The system is not an oracle. It emits typed, grounded, governed artifacts whose route, evidence, and receipt boundaries can be inspected before downstream use. SONYA-AEGIS-SMOKE-02 proves a local deterministic governed episode through the Sonya local membrane with verified direct-call blocking. The WAVE Gold-Physics family provides closed-form metric calibration. The experiment suite registry and reproducibility pack preserve claim boundaries across accepted, partial, blocked, and planned phases. Claims are bounded to auditability, route disposition, artifact traceability, and non-authority guardrails; this draft is not truth certification, not deployment authority, and not final answer release.
