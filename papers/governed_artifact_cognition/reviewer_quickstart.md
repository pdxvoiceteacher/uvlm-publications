- Governed artifact cognition can improve auditability by routing model work through typed candidate, ingress, evidence, route, receipt, inventory, manifest, export-parity, and reproducibility artifacts.
- SONYA-AEGIS-SMOKE-02 provides accepted local deterministic fixture evidence with `execution_path = sonya_local_membrane` and verified direct-call blocking.

## How to reproduce SONYA-AEGIS-SMOKE-02

In a local CoherenceLattice checkout, run the SONYA-AEGIS-SMOKE-02 harness documented by `docs/SONYA_AEGIS_SMOKE_02.md`. The expected evidence posture is accepted local fixture only, direct-call blocking verified, and `execution_path = sonya_local_membrane`.

```bash
cd ../CoherenceLattice
python -m pytest -q \
  python/tests/integration/test_sonya_aegis_smoke_02_acceptance_harness.py \
  python/tests/integration/test_sonya_aegis_publisher_boundary_finalizer.py
```

On Windows, use the accepted harness path:

```powershell

  -CiMode
```

## How to inspect the repro pack

Inspect the JSON pack, acceptance matrix CSV, artifact manifest, Markdown report, and zip bundle produced by `python -m coherence.tools.build_experiment_suite_repro_pack`. Confirm that accepted, partial, blocked, and planned phases are represented and that blocked claims and `does_not_test` statements remain visible.

## How to inspect WAVE family packet

Open `waveform_gold_physics_family_acceptance_packet.json` generated by `python -m coherence.waveform.family_acceptance` and the `docs/WAVE_GOLD_PHYSICS_FAMILY.md` primer. Confirm that WAVE-00R through WAVE-03R are bounded to closed-form calibration and do not claim psychoacoustics, universal ontology, AI cognition, or deployment readiness.

## How to check non-authority guardrails

Look for explicit guardrails stating that receipts, routes, memory intent, Publisher candidates, and reproducibility packs are non-authoritative. Required phrases include not truth certification, not deployment authority, not final answer release, local fixture only, and requires external peer review.

## How to read route dispositions

Read route dispositions as governance posture, not authorization. A route can send an artifact to human review, quarantine, rejection, or bounded candidate handling. It does not authorize memory writes, prior canonization, deployment, truth certification, or final-answer release.
